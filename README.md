# 👻 HYENZY-X ANIME & GAMES SCRAPER

> **Multi-Source Scraper • Anime4K Upscale • Playwright Core • Discord RPC**

---

## 🇺🇸 English

### 🌟 Overview
**Hyenzy-X** is a powerful, all-in-one media scraper and player designed for anime, movies, and games. It leverages **Playwright** for robust web scraping and **MPV** with **Anime4K shaders** to provide a high-quality viewing experience with real-time upscaling.

### 🚀 Features
- 🔍 **Multi-Source Scraping**: Automatically find content across multiple providers for animes, movies, and games.
- 📺 **High-Quality Playback**: Integrated MPV player with **Anime4K** shaders for real-time 4K upscaling.
- 🎮 **Game Downloader**: Dedicated system for searching and downloading games from various sources.
- 🎭 **Discord Rich Presence**: Show your friends what you're watching or playing with custom RPC.
- ⚡ **Fast & Efficient**: Optimized scraping logic with caching support.
- 🎨 **Modern TUI**: Clean and intuitive Terminal User Interface.

### 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Pedrohs1771/Hyenzy-X-Anime-Scraper
   cd Hyenzy-X-Anime-Scraper
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   playwright install chromium
   ```

3. **Requirements:**
   - [Python 3.10+](https://www.python.org/)
   - [MPV Player](https://mpv.io/) (Added to PATH)

### 📖 How to Use
Run the main launcher:
```bash
python main.py
```
Follow the on-screen menu to choose between Animes, Movies, or Games.
